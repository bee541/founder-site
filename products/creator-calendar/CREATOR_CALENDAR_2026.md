# Creator Calendar 2026 — Your 12-Month Publication OS

**A full year, structured in 4-week sprints or 13-week quarters. Notion-ready. Printable-ready. Google Calendar-ready.**

> Each quarter has a theme, each month has a focus, each week has ONE priority. 
> Burnout is not a feature. Efficiency without progress is not a goal.

## How to Set Up in Notion

1. Import `notion_import.csv` as a new database table
2. Add columns: `Status` (Not Started / In Progress / Done), `Revenue` (EUR), `Audience Growth`, `Notes`
3. Set a monthly repeating calendar event for the check-in review
4. Duplicate this page for every new year or project — the system scales
